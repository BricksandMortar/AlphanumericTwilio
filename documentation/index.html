What's the Plugin About?

In certain countries it's possible to send messages using an alphanumeric sender id. If your region is supported then this plugin let's you send using an alphanumeric id from Rock. It's a great way to use your brand to increase engagement and is suited to large communication blasts as it prevents replies.

Note: This Twilio functionality is not available in the United States or with trial Twilio accounts.

Setup

Twilio

Visit the Twilio console and note down your Account SID and Auth Token.

Rock

To setup the plugin you'll need to enable the Alphanumeric SMS Twilio communcation transport and the Alphanumeric SMS communication medium.

Communication Transport

Visit Admin Tools > Communications > Communication Transports and select the Twilio Alphanumeric SMS transport. Paste in your Account Sid and Auth Token and then change the transport to active. The optional footer is appended to all simple mode communications and optionally when selected when sending a communication using the full mode. Use this to provide a way for your users to opt out or to contact the sender. This field has access to the Lava context of the person the message is being sent to and the sender.

Communication Medium

Visit Admin Tools > Communications > Communication Mediums and select the Alphanumeric SMS medium.Change the medium to active and then select the Twilio Alphanumeric SMS Transport.
